Introduction

This project is a recreation of my first coded mobile site to responsive site. To achieve the responsivnes, I have learnt to use CSS-grid and Media queries.

My website consist of the small school tasks, such as moodboards, logo and explanation of my design choices.

Task

Create the responsive version of the mobile site. Use at least 1 break point and CSS-grid.

Approach

Attending “learning-by-doing” presentations of teachers in class, using help of tutor (Thomas Lehoczky) outside of class, following HTML and CSS tutorials on Lynda.com

Challenges

To use CSS-grid is still challenging for me. It will be the area I will strongly focus on in the future. My goal is to feel comfortable to use them.

Tools

Brackets (code editor), Developer Tools (Chrome), Cyberduck (FT)

Purpose

Learn about making your website more responsive to different viewports and devices. Use More advanced HTML5 and CSS3 for creating responsive web design. Including media queries, responsive navigation, advanced selectors

Behind
Design

In this case I was focusing mainly on functuality of the website, not the design. The most important lesson for me was to adjust website with grids to different viewports such as desktop, tablet, mobile.

The background color of the website change for the better visual differentiation of responsivness from desktop to tablet and mobile.
