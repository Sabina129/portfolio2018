Introduction

JavaScript is a powerful scripting language, widely used for controlling web page behaviour. Learning JavaScript allows me to level up my possibilities of creating creative and more user friendly websites. I can move from creating just the style of the website to its behaviour. I feel like a God, woho!

In this task I am experimenting with JavaScript in the funny way. I have a Ninja character which you can control by clicking on buttons. Try it!

Task

Building a webpage containing a character with 10 buttons When the user clicks a button, the character starts a specific animation. For example: jump.

Approach

I was able to complete my task by being present on “learning-by-doing” presentations at school.

Challenges

Combine CSS with JavaScript (When the button is clicked, call a function that adds a class to the character (thereby animating it)).

Tools

Brackets, Lynda.com

Purpose

Be able to apply theories and methods within systems development Theories and methods used include, pseudo-code and sequence diagrams.
