Introduction

I have chosen to introduce you this project in order to give you the opportunity to see my first attempt of designing my portfolio website. I am using this project to see my progress.

In the XD prototypes are placed my first moodboards, logo made for my first website and explanation of design choices.

Task

Create paper prototype in real sizes for mobil, tablet and desktop. Create style tile of my webdesign. Make interactive prototype in user experience design software “Adobe XD”. Describe the design principles in use.

Approach

I was trying to avoid the situation where I create the website which just I can understand. Therefore, I was trying to conduct a user testing with as many people as possible.My goal was to gain relevant feedbacks in order to be able to design user friendly website and grow as a designer.

Challanges

Applying design principles for desktop and mobile version of the website. Trying to create website which works not just for me but also for users.

Tools

Paper and pen, Youtube tutorials , Adobe XD.

Purpose

Learn about basic design principles, various types of prototyping, and testing your designs to ensure user friendly web designs.

Behind
Design

Frontpage picture of the mobile version visualy represents the way to go deeper. On the desktop version I experimented with the picutre of mountains, because it represents mental "rising" and improvement.

Contemporary, which is connected with minimalism, was my predefined theme for this task. Because of that, I placed my focus to create the whole concept as combination of the white space and color contrasts. My color palette is chosen from the research of articles about contemporary style.

Sizes of the ellipses on the menu page are made by its importance. From my user testing I observed, that the first choice of a user is to find and see my work. For this reason the ellipse My work is the biggest one. The journey of the user is finishing with the contact - this ellipse is the smallest.

Composition of every page is mainly based on symetry, because I wanted to create coherence.
